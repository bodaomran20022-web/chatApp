# 💬 Real-Time Chat App

A cross-platform real-time messaging app built with Flutter & Firebase.

## ✨ Features
- Real-time messaging with Firebase Firestore
- Secure authentication with Firebase Auth
- WhatsApp-inspired chat UI with smart bubble alignment
- Auto scroll to latest message
- Keyboard-safe responsive layout

## 🛠️ Tech Stack
- Flutter & Dart
- Firebase Authentication
- Cloud Firestore (Real-time Streams)
- StreamBuilder
